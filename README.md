# 📊 Google Play Store Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Google Play Store dataset to uncover meaningful insights about app performance, user engagement, pricing, installs, ratings, reviews, and categories.

Using Python and popular data analysis libraries, the project cleans the data, visualizes trends, and answers business-oriented questions through statistical analysis and charts.

---

## 📂 Dataset

The project uses the following datasets:

- `googleplaystore.csv`
- `googleplaystore_user_reviews.csv`

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📈 Analysis Performed

- Data Cleaning and Preprocessing
- Missing Value Analysis
- Category-wise App Distribution
- Free vs Paid Apps Analysis
- Ratings Distribution
- Reviews Analysis
- Install Count Analysis
- Price Analysis
- Revenue Potential Estimation
- Correlation Analysis
- Content Rating Analysis
- Genre Analysis
- App Size Analysis
- Last Updated Trend Analysis
- Success Index Calculation

---

## 📊 Key Insights

- Identified the most popular app categories based on installs and ratings.
- Compared free and paid apps across ratings and user engagement.
- Analyzed the relationship between app price, installs, reviews, and ratings.
- Estimated revenue potential across different app categories.
- Explored update trends and category-wise performance.

## 🚀 How to Run

1. Clone the repository

```bash

```

2. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open the notebook

```bash

```
